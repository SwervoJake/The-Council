# AI Agent Council — Adaptive Kaizen Decision Engine

A multi-agent advisory system for continuous improvement across life domains. Five personas (Neo, Bruce, Cena, Luffy, Professor) provide structured debate, synthesis, and actionable recommendations under a Kaizen philosophy.

## Quick Start

1. Open this project in Cursor.
2. **Open `Board-Room.md`** — your pinned council interface.
3. Right-click the Board-Room tab → **Pin Tab** to keep it always available.
4. Run a council session: *"Professor, initiate council session. Topic: [your question]."*
5. Review the output: Top 3 decisions, one experiment, one habit adjustment.
6. Approve tasks before they are logged.

## Philosophy

- **Kaizen:** Small compounding improvements
- **Max 3 actions per session**
- **User is Sovereign** — no unapproved execution
- **Power earned, not granted** — tier progression over months
